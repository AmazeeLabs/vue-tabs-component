The tab component above is powered by [spatie/vue-tabs-component](https://github.com/spatie/vue-tabs-component). The component will use the fragment of the url to choose which tab to open. So if you navigate to [http://vue-tabs-component.spatie.be/#second-tab](http://vue-tabs-component.spatie.be/#second-tab) the contents of tab named `Second tab` will be displayed. The component will also [remember which tab was opened previously](https://github.com/spatie/vue-tabs-component#remembering-the-last-opened-tab). So if you click `Second tab` and than navigate to [http://vue-tabs-component.spatie.be](http://vue-tabs-component.spatie.be) the second will be displayed.
